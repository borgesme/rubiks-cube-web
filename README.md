## 主要功能摘要

- 27 个独立 Cubie（间隙 0.06）、Canvas 程序化圆角贴纸纹理（黑边+高光）、PCFSoft 阴影 + 双光源
- 世界坐标动态筛层 + Pivot attach/detach + 坐标取整清洗（审查中经 2 万步模拟验证零发散）
- 投影向量手势识别、带符号半径方向修正（含缝隙内壁命中场景）、1:1 跟手、磁吸回弹
- 多触点过滤、手势期间禁用视角控制、Scramble（20 步）/ Reset 按钮
